# Password Generator

A simple **React + TypeScript** password generator with adjustable length and character options.

## Features

- Generate secure passwords with:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Set password length (6–12 characters)
- Password strength indicator (Low → Very Strong)
- Copy password to clipboard with a single click
- Interactive and responsive UI

## Usage

1. Clone the repository:

```bash
git clone https://github.com/Rohozhnyi-Illia/Password-generator.git
cd Password-generator
```

## Install dependencies

```bash
npm install
```

## Start the development server

```bash
npm start
```

Open http://localhost:3000 to view the app in the browser.
